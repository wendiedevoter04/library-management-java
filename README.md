# Library Management System (Java)

This is a simple Java project I built to practice Object-Oriented Programming concepts by modeling a basic library system.

The goal was to understand how classes, objects and data structures work together in a small real-world scenario.

## Features

- Add books to the system
- Store books using an ArrayList
- Display all available books
- Track basic book availability

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList

## How It Works

The system uses Java classes to represent books and manage them in a collection.  
Books are stored in an ArrayList, and basic operations like adding and displaying books are handled through methods.

## Sample Data

Includes sample entries from Kenyan authors such as Ngũgĩ wa Thiong'o and Yvonne Owuor.

## What I Learned

While building this project, I learned how to:
- Structure a program using classes and objects
- Work with ArrayLists to store and manage data
- Apply basic OOP principles in a practical project
