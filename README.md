# Library Management System (Java)

A simple Java-based library system demonstrating Object-Oriented Programming concepts.

## Features
- Add books
- Store books using ArrayList
- Display all books
- Basic book availability tracking

## Technologies
- Java
- OOP (Classes & Objects)
- ArrayList

## Sample Data
Includes Kenyan authors such as Ngũgĩ wa Thiong'o and Yvonne Owuor.

## Purpose
Built as a learning project to understand Java OOP and system design basics.
